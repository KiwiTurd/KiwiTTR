# 🏓 KiwiTTR

KiwiTTR is a modern Table Tennis Rating System designed for New Zealand clubs.

## Features

- Player Management
- Club Management
- Events
- Match Recording
- Automatic Rating Calculations
- Rankings
- Rating Simulator

## Planned Features

- Player Profiles
- Rating History Graphs
- Tournament Management
- Public Rankings
- Club Rankings
- Match History
- Head-to-Head Records

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Local Storage (currently)

## Development

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build

```bash
npm run build
```
